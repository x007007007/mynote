超速学习
------

#. 后设学习
    由终点确定起点

#. 直接学习
    * 针对最后你想使用方式学习
    * 省略中间非必要技能

#. 如何找到练习的点
    * 连贯法
        找到问题，解决问题，发现新问题，迭代
#. 如何应对遗忘
    笔记中记录问题，而非答案，

#. 如何确定学会了
    费曼技巧
    如何别人理解
