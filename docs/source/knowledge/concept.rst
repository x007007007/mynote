哲学概念
=======


.. glossary::

    心流
        英语：（Flow），也有别名以化境（Zone）表示，亦有人翻译为神驰或沉浸状态
        定义是一种将个人精神力完全投注在某种活动上的感觉；心流产生同时会有高度的兴奋感及充实感等正向情绪。

        * 我们倾向去从事的活动
        * 我们会专注一致的活动
        * 注意力投入在当前活动
        * 动作与意识的合并
        * 反省自我意识的丧失
        * 有清楚目标的活动
        * 有立即回馈的活动
        * 我们对环境与动作有主控感--挑战与技能之间达成平衡
        * 在从事活动时我们的忧虑感消失
        * 主观的时间感改变--例如可以从事很长的时间而不感觉时间的消逝

        当人们处于心流状态，可能会出现四个特征：

            #. 自动运转：事情做起来顺手不需多加思考，身体自动发挥。
            #. 时间流逝：处于心流状态中，不会在意时间的流逝，直到回到正常状态后，才会注意到已经过了多长时间。
            #. 不觉他物：专注投入事物之中，导致不易察觉像是饥饿、手机震动等感觉与刺激。
            #. 感到愉悦：在事情完成后，感受到愉悦、满足、成就感等正向情绪。
